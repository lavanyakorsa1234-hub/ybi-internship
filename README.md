# AdaBoost-LSTM: Credit Card Fraud Detection System

## 📌 Project Overview
This project presents a hybrid deep learning and ensemble learning framework for detecting fraudulent credit card transactions using an AdaBoost-LSTM model. The system combines the sequential learning capability of Long Short-Term Memory (LSTM) networks with the boosting strength of AdaBoost to improve fraud detection accuracy. The proposed model effectively captures temporal transaction patterns and focuses on difficult-to-classify fraudulent cases, resulting in high accuracy and robust performance for real-world fraud detection applications.

## 🚀 Features
- Hybrid AdaBoost + LSTM architecture
- Sequential transaction analysis
- Fraud and non-fraud classification
- Data preprocessing and normalization
- Handling imbalanced datasets
- High accuracy and recall
- ROC and Precision-Recall curve visualization
- Performance comparison with machine learning algorithms
- Real-time fraud prediction capability

## 📂 Dataset
The dataset contains credit card transaction records consisting of legitimate and fraudulent transactions. It includes anonymized features, transaction amount, time attributes, and class labels for fraud detection.
https://www.kaggle.com/datasets/mlgulb/creditcardfraud?utm_source=chatgpt.com
